# JUC & JVM

## 模块

### juc

初学 Java 并发编程与高并发解决方案，包含各类教学案例、企业级案例、个人实践案例

- annotations：定义了四类注解，推荐 / 不推荐写法，线程安全 / 不安全写法
- demo：Java 并发练习案例
- example：Java 高并发与并发编程学习主目录
- interview：并发编程面试个人总结

### jvm

JVM 学习工程，记录了 JVM 学习笔记、案例

### data-structure

数据结构与算法学习工程

### simple-seckill

简单的单机秒杀系统
